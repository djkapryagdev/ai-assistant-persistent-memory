# AI Assistant with Persistent Memory

A conversational AI assistant built in Python with persistent memory between sessions.

## Features

- Persistent memory stored locally in JSON format
- Retains user preferences and context across sessions
- Adaptive behavior based on time of day
- Integration with large language model APIs

## Tech Stack

- Python
- JSON (local storage)
- LLM APIs (generative AI)

## Project Structure

main.py          # Main assistant logic

memoria.json     # Local memory storage (not included in repo)

conversaciones/  # Session logs (not included in repo)

## About

This project explores the concept of an AI companion with emotional presence and contextual awareness. Built independently as a personal research project focused on human-AI interaction and persistent state management.
