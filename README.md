# Fara Ops — Personal Intelligence Assistant

A local productivity and research assistant built in Python.
Tracks AI developments, manages reminders, saves notes and helps you stay organized.

## Features

- Persistent local memory (JSON)
- Reminders and alarms (active while program runs)
- AI watchlist — track ChatGPT, Gemini, Mistral, Llama and more
- Personal notes and improvement ideas with timestamps
- Session history log saved automatically
- Time-of-day awareness
- Safe defensive security checklist
- Safe read-only folder review (no file execution or modification)

## Tech Stack

- Python 3.8+
- JSON (local storage)
- Standard library only (no external dependencies)

## Project Structure

fara_ops.py          # Main assistant logic

data/

fara_memory.json   # Local persistent memory (not included in repo)

sessions/          # Session logs (not included in repo)

## Usage

```bash
python fara_ops.py
```
